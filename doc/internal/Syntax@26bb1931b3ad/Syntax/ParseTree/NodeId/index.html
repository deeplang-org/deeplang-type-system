Module ParseTree.NodeId

NodeId contains types for unique ids for various kinds of AST nodes. Can be used to attach extra data to AST nodes, via an external table.

type pattern =
  1. | PatId of int
type expr =
  1. | ExprId of int
type stmt =
  1. | StmtId of int
type impl =
  1. | ImplId of int
